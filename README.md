# password-strength-checker
This program is a simple password strength checker that validates requirements and scores password strength. First cybersecurity type of coding project, built using python. This program requires Python 3 and should be run using password_strength_checker.py in the terminal. This program gives the user a set of requirements for a password. Requirements include password to be between 8-16 characters in lenth, contain a special character, contain a digit, contain a capital letter, and not contain any spaces. When the password is entered, if it violates any of the restrictions, error messages will be returned informing the user what requirements were not satisfied. If no requirements are violated, the password is given a score based on its length, number of digits, and number of special characters, and the higher score the password recieves, the higher strength it is. Through this project I strengthened my knowledge on how to break down problems into smaller steps and how to use a flag variable to determine an end result. Overall, this project was a great way to get thinking about cybersecurity and specifically, how password security works in real situations. In the future I would like to improve this code by allowing the user to input multiple passwords without restarting, and one day integrate this code with Have I Been Pwned API in order to give users better feedback on the strength of their password.
